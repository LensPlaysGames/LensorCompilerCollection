* Out-Of-Order Declaration

Glint supports out-of-order function declaration.

Glint also ignores declarations when it comes to picking an implicit return value from the top level function.

#+NAME: source
#+begin_src glint
  foo 42;

  foo : int(x : int) = x + 27;
#+end_src
